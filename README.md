# check_docx_tilte

## 功能

- [X] 修改每个文档的第一段话后+第x期（会破坏原有的样式）
- [X] 合并每个word文档的第一段话和第二段话
- [X] 输出每个word文档第一段话和第二段话的在/output.txt
- [ ] 把正文的第一句话改为标题级别
- [ ] 合并所有word到一个words

## 用法

1. 在当前环境下安装 `python-docx`库：pip install python-docx
2. 在根目录下运行：python check_docx_title.py
